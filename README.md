<h1>Juego Ahorcado Assembler</h1>
<p>El programa implementa el juego del ahorcado en ensamblador para Linux. Selecciona una palabra, muestra el ahorcado por etapas y permite ingresar letras, indicando aciertos o errores con colores. Ganas al adivinar todo y pierdes tras seis fallos.</p>
